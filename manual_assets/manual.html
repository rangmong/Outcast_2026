💾 데이터 백업 & 복원
📤 데이터 백업하기
  • 1 하단 ⚙️ 설정 탭으로 이동
  • 2 💾 데이터 관리 섹션 찾기
  • 3 백업 버튼 탭 → JSON 파일 다운로드!
파일명 예시: outcast_backup_2026-05-04.json
📥 데이터 복원하기
  • 1 하단 ⚙️ 설정 탭으로 이동
  • 2 💾 데이터 관리 섹션에서 복원 버튼 탭
  • 3 백업해 둔 .json 파일을 선택
  • 4 확인 버튼을 누르면 데이터가 복원돼요! 🎉
⚠️ 주의사항

복원 시 현재 데이터가 덮어쓰여져요!

복원 전에 먼저 백업을 해두는 게 안전해요.

내역서 이미지는 별도로 저장되므로
복원 후 이미지는 다시 첨부해야 할 수 있어요.
